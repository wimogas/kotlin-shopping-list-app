# Shopping List App

## Description

Simple Shopping List Android App made with Kotlin, ROOM, Compose and Clean Architecture MVVM

## Features

- [x] Add new items to your list
- [x] Add quantities
- [x] Edit the items
- [x] Mark as Bought
- [x] Delete items
